# Custom Settings

### System

| Shortcut  | Action |
|---|---|
| Super + h | Desktop Left |
| Super + l | Desktop Right |
| Super + j | Window Right |
| Super + k | Window Left |
| Super + , | Window to Desktop Left |
| Super + . | Window to Desktop Right |
| Super + ; | Launch Konsole |
| Super + m | Maximize Window |
| Alt + Space | Synapse Launcher |

### Konsole

| Shortcut  | Action |
|---|---|
| Alt + p | paste |
| Alt + y | copy/yank |
| Alt + j | next tab |
| Alt + k | previous tab |
| Alt + h | new tab |
| Alt + l | close tab |
| Alt + , | move tab left |
| Alt + . | move tab right |
